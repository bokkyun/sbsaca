  --color-button-hover:   #C45C63;  /* 버튼 호버 배경 */

  --color-link-hover:     #F16D6C;  /* 링크 호버 색상 */

## 컬러 사용 가이드

### Primary — `#F16D6C` 코랄 레드
- CTA 버튼, 뱃지, 태그 배경
- 히어로 섹션의 포인트 요소 (아이콘, 언더라인)
- 진행 바, 차트 등 데이터 시각화 강조색
- **주의**: 흰 배경 위에서 명도 대비 충분 — 본문 텍스트보다는 UI 요소에 사용 권장

### Primary Dark — `#C45C63` 딥 로즈
- 버튼 호버/눌림(active) 상태
- 링크 텍스트 기본 색상
- 카드 보더, 구분선 포인트

### Secondary — `#333535` 차콜
- 모든 제목(h1~h4) 및 본문 텍스트
- 네비게이션 바 배경 (라이트모드)
- 푸터 배경, 다크 카드 배경

### Background — `#FFF9F6` 웜 화이트
- 페이지 전체 메인 배경
- 카드, 모달 배경
- 순백(`#FFFFFF`) 대신 사용해 따뜻하고 부드러운 느낌 연출

### Background Secondary — `#F2EDE9`
- 교차 섹션 배경 (홀수/짝수 섹션 구분)
- 인풋 필드, 텍스트 에어리어 배경

### 모바일 고려사항
- 버튼 최소 터치 영역 44px 확보, `--color-button-primary` 배경에 `#FFF9F6` 텍스트 조합 권장
- 코랄 계열은 소면적 포인트로만 활용 — 넓은 배경 사용 시 피로감 유발 가능
- 다크모드에서 `#F16D6C`는 채도가 높아 눈에 잘 띄므로 아이콘/링크에 적극 활용

## 디자인 특징

이 팔레트는 **모던 미니멀 + 따뜻한 감성**의 조합입니다.

- **차콜(`#333535`) + 웜 화이트(`#FFF9F6`)** 베이스가 순수한 흑백보다 부드럽고 인간적인 느낌을 줌
- **코랄-로즈 계열** 포인트는 에너지와 친근감을 동시에 전달 — 패션, 뷰티, 크리에이티브 포트폴리오에 잘 어울림
- 전체적으로 **따뜻한 뉴트럴 + 생동감 있는 포인트** 구조로 브랜드 신뢰감과 개성을 함께 표현
